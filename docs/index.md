## Матрица компетенций

=== "Инженерия и автоматизация"
    * **Python (Advanced):** Разработка отказоустойчивых парсеров и конвейеров данных (ETL), автоматизация сбора информации из защищенных сред (Selenium в Headless-режиме, BeautifulSoup4, Requests, многопоточность, Pandas).
    * **Извлечение данных:** Точный захват изолированных бизнес-констант и метрик (RTP, волатильность) с помощью сложных **XPath-выражений** (поиск по текстовым предикатам, осям предков/потомков) и CSS-селекторов в условиях нестабильной корпоративной верстки.
    * **Проектирование алгоритмов:** Создание сквозных автоматизированных процессов, реализация логики конечных автоматов (State Machines).
    * **Валидация данных:** Реверс-инжиниринг документов, парсинг и контроль целостности сложных вложенных структур (HTML / XML / JSON).
    * **Инфраструктура:** Реализация методологии Docs-as-Code, интеграция с корпоративными REST API, настройка CI/CD для доставки контента.

=== "Анализ данных и NLP"
    * **Мультиязычная обработка:** Токенизация и лемматизация в условиях строгих лингвистических ограничений (обработка беспробельного тайского текста, интеграция `Stanza` + `CAMeL Tools` для арабского, работа с `UzMorphAnalyser` для узбекского и `Malaya` для малайского языков).
    * **Семантический векторный анализ:** Применение LLM и энкодеров Hugging Face (`LaBSE`, `RoBERTa`) для математического расчета смыслового сходства между разноязычными текстами и выявления аномалий.
    * **Синтаксический анализ:** Использование графов зависимостей (`depparse`) для контекстного вычленения терминов, автоматизированной проверки глоссариев и контроля регистра.
    * **Автоматизированный контроль:** Проектирование конвейеров обратного перевода (Back-translation) со сквозной генерацией наглядных отчетов (Diff) для быстрого машинного и ручного аудита больших массивов данных.

=== "Системный анализ и архитектура"
    * **Системный и бизнес-анализ:** Сбор и формализация требований, декомпозиция сложной игровой логики (расчет RTP, волатильности, схем выплат), проектирование алгоритмов взаимодействия и сквозного маппинга данных **с использованием нотаций визуального моделирования (UML)**.
    * **Регуляторное соответствие:** Проектирование и аудит ИТ-решений под строгие международные стандарты (аудит тестовых лабораторий **GLI / BMM Testlabs**, лицензирование MGA / UKGC), эквивалентные по уровню ответственности стандартам MedTech-отрасли.
    * **Управление рисками и техдолгом:** Проактивный анализ уязвимостей архитектуры (Bus Factor, накопление технического долга, изменение структур данных на стороне партнеров), разработка стратегий снижения рисков и планов перехода на API-контракты.
    * **Эргономика и проектирование:** Реализация концепции декларативного управления (Configuration-as-Code) вместо избыточных графических интерфейсов. Разработка интерактивных руководств по устранению неполадок для снижения когнитивной нагрузки на пользователей.
